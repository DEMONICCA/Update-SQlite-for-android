> **DESCRIPTION...**    
> This Magisk module provides a statically linked and stripped sqlite3 binary for arm64-v8a, armeabi-v7a, x86, and x86_64 devices. [Source](https://www.sqlite.org/)
<hr/>

> [!TIP]
> 1. Open Magisk Manager or Ksu or APatch
> 2. Install module [SQLite Multi arch 3.47.0](https://t.me/modulkuntul)
> 3. Reboot
<hr/>

> [!NOTE]
> - Support Android 10-15
> - Support [Ksu](https://github.com/tiann/KernelSU/releases)-[APatch](https://github.com/bmax121/APatch/releases/tag/10763)-[Magisk](https://github.com/topjohnwu/Magisk/releases/tag/v28.0)
